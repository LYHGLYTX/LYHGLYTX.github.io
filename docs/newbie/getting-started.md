# 准备开始

## 下载游戏

CCB 的发布版本可以在 [GitHub Releases](https://github.com/LYHGLYTX/Cataclysm-Cleanwater-Bomb/releases) 获取。

| 平台 | 说明 |
|---|---|
| Windows | 下载 `.zip`，解压后运行 `cataclysm-tiles.exe` |
| Linux | 下载对应包，解压后运行 `cataclysm-tiles` |
| Android | 下载 `.apk` 安装 |

## 第一次启动

启动后建议先做这几件事：

1. **设置语言**：`Options → General → Language` 选择「简体中文」
2. **检查字体**：中文显示异常时在 `Options → Graphics` 调整字体
3. **加载贴图**：`Options → Graphics → Tileset` 选择你喜欢的贴图包

## 基础按键

| 按键 | 作用 |
|---|---|
| 方向键 / `hjkl` | 移动 |
| `i` | 查看物品栏 |
| `e` | 吃东西 |
| `g` | 捡起物品 |
| `?` | 帮助 / 按键列表 |
